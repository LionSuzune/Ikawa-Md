```bash

$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ cd /sdcard
$ cd Ikawa-Md
$ npm i

$ node .
```

## TERMUX WITH UBUNTU

```bash
apt update && apt full-upgrade
apt install wget curl git proot-distro
proot-distro install ubuntu
echo "proot-distro login ubuntu" > $PREFIX/bin/ubuntu
ubuntu
```
